Welcome!!!!

This data analysis project consisted on creating a random list of 500+ cities, using that list to make a series of calls to OpenWeather API and use that information to answer the big question:

As we move closer to ecuator do temperatures get higher?

Then, from that cities list and filtering by weather conditions I made a series of API calls to Google Places in order to obtain a humidity heat map, then placing markers with hotel information close by the city geographical center
